# Claude Code

Este proyecto usa harness-cli. Para entender cómo trabajar aquí:

1. Lee primero `HARNESS.md` (contrato del harness).
2. Si existe `AGENTS.md`, sus reglas son obligatorias.
3. Si dudas en cómo operar, consulta `INSTRUCCIONES.md`.
4. Para los prompts más comunes, mira `FAST-USAGE.md`.

Los agentes están disponibles en `.claude/agents/` (descubrimiento nativo de Claude Code).
